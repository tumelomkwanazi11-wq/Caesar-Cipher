This project implements a Caesar Cipher encryption and decryption system using Python. The Caesar Cipher is one of the earliest and simplest methods of cryptography, where each letter in a message is shifted by a fixed number of positions in the alphabet.

The program allows users to both encrypt and decrypt text by specifying a shift value. It includes input validation to ensure that the shift value is within a valid range and demonstrates how text transformation can be automated using Python‚Äôs built-in string translation methods.

The system works by shifting each letter of the alphabet by a specified number of positions.

Encryption Process

The user provides text and a shift value.

The alphabet is shifted according to the given value.

A translation table is created.

The original text is converted into encrypted text.

Decryption Process

The encrypted text is provided along with the same shift value.

The shift direction is reversed.

The original message is restored.

The program supports both lowercase and uppercase letters while keeping non-alphabet characters unchanged.
